Penetration-testing write-ups for **Hack The Box** and **TryHackMe** machines.
Each write-up documents the full kill chain — enumeration, vulnerability
discovery, exploitation, and privilege escalation — using a consistent template.

> All targets are intentionally vulnerable lab environments. Flags are
> intentionally omitted; content is for educational purposes only.

## Write-up Structure

Every machine follows the same template:

- **Summary** — one-paragraph overview of the attack chain
- **Machine Information** — name, difficulty, OS, platform
- **Attack Path** — numbered high-level steps
- **Reconnaissance** → exploitation sections
- **Vulnerability Analysis** — root cause + fix per finding
- **Tools Used** / **Key Takeaways**
